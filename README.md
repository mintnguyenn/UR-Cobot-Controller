# UR Manipulator Robots Controller

This repository contains a ROS software developed for controlling manipulator robots in the UR series, such as UR3(e), UR5(e) and UR10(e). The software facilitates communication with the robot via ROS and can control either the physical UR robot or the simulation one in Gazebo.
